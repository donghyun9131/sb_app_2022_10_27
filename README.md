# CRUD 웹 게시판 제작하기

1인 프로젝트 'CRUD 웹 게시판 제작하기' 입니다.

Spring Boot + Mybatis 를 이용하여 웹 페이지를 구현했습니다.

현재는 간단한 CRUD 컨트롤러와, 이를 요청 / 응답 받는 VIEW 페이지가 구현되어 있습니다.
